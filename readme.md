# num2Han

> 阿拉伯数字转化为汉字（简/繁体）

## 安装

`npm i -S num2han`

## 代码示例

```typescript
import num2Han from "num2han";

num2Han.convert(123); // 一百二十三

num2Han.setMode("繁");

num2Han.convert("123"); // 壹佰贰拾叁
```
